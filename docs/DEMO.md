# Demo Guide

Run locally:

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4188`.

Try:

- Click **Inspection** to move to an inspection pose.
- Click **Reach fixture** to simulate a target reach.
- Click **Safe fold** to fold the arm into a safe home position.
- Type a goal in the Agent Console and click **Plan & apply**.
- Click **Copy JSON snapshot** to export the full twin state.

The demo intentionally uses generic synthetic state and does not connect to hardware.
