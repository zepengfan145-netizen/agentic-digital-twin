# Security Policy

## Supported versions

The `main` branch is the supported development branch for now.

## Reporting

Please do not publish sensitive facility, CAD, device, network, or credential details in public issues. Open a private advisory when available, or use the maintainer contact path listed by the repository owner.

## Demo boundary

The public demo is a static simulation. It must not be connected directly to physical equipment without an independent safety layer, authentication, authorization, and hardware-specific emergency stop procedures.
